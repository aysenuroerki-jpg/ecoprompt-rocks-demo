Bölüm A · 1/2

Sistem Mimarisi

Sistemin can alıcı noktası şu: ROCKS analizi kullanıcının cihazında çalışır. Çocuğun ChatGPT/Gemini'ye yazdığı gerçek prompt hiçbir zaman EcoPrompt sunucusuna gönderilmez — sadece analizin sonucu (kaç ml su, hangi ROCKS bileşeni eksik, kaç Eco-Coin) anonim bir metrik olarak yukarı akar. Bu hem 10–14 yaş grubu için DSGVO/GDPR uyumluluğunu kolaylaştırır hem de okulların "çocuğumun promptları nereye gidiyor" sorusuna net bir cevap verir.

İstemci katmanı — tarayıcı eklentisi

Manifest V3 content script, chatgpt.com, gemini.google.com, claude.ai gibi izin verilen alan adlarına enjekte edilir. Prompt kutusundaki submit olayını (Enter tuşu / gönder butonu) yakalar, metni gönderim öncesinde okur, yerel olarak ROCKS analizini çalıştırır ve Shadow DOM içinde Robby'nin kartını gösterir — böylece EcoPrompt'un stilleri ChatGPT'nin arayüzüyle çakışmaz.

İki entegrasyon modu

Mod 1 — Eklenti (pasif gözlem)

Çocuğun zaten kullandığı kişisel ChatGPT/Gemini hesabına dokunmadan, sadece tarayıcıdaki metni okur. API anahtarı gerekmez, kurulumu tek tıkla yapılır. 13–14 yaş grubu için uygun; okul dışı kullanımı da kapsar.

Mod 2 — Okul API vekili (proxy)

Okulun sağladığı OpenAI/Gemini anahtarı EcoPrompt ara katmanından geçer. Bu modda zayıf bir prompt gönderilmeden önce Robby "önce düzeltelim mi?" diye sorabilir — 10–11 yaş için daha korumalı bir "eğitim tekerlekleri" deneyimi.

Backend servisleri

  • Auth — okul tarafından yönetilen takma ad hesapları; 16 yaş altı için veli onay akışı (DSGVO Md. 8 mantığı), gerçek isim/e-posta zorunlu değil.
  • Ingest API — anonim metrik olaylarını doğrular, kullanıcı başına hız sınırı koyar (aynı prompt'u art arda göndererek Eco-Coin "farm" etmeyi engeller).
  • Garden State Service — her olay bir event log kaydıdır; bahçe durumu (ağaç büyümesi, nehir berraklığı, coin bakiyesi) bu log'un indirgenmiş görünümüdür — bu sayede "neden nehrim kirlendi" sorusu her zaman geri izlenebilir.
  • Leaderboard Service — okul/Bezirk bazında haftalık materialized view; near-real-time güncellenir, her Pazartesi sıfırlanır.
  • Robby Content Service — ROCKS eksik bileşeni → lokalize mascot repliği eşlemesi; öğretmen panelinden mesaj tonu ayarlanabilir.
Gizlilik notu — Prompt metninin kendisi hiçbir zaman ağ isteğine dahil edilmez; yalnızca sayısal analiz sonucu senkronize edilir. Bu, hem çocuk verisi koruması hem de okulların yapay zeka sağlayıcılarıyla (OpenAI, Google) olan kullanım şartları çakışmalarını en aza indirir.

Bölüm B

ROCKS Analiz Motoru

Aşağıdaki fonksiyon saf JavaScript'tir (React içinde de kullanılabilir), harici bir NLP servisi çağırmaz — anahtar kelime eşleşmesi (substring) ve iki basit regex'e (rakam, tırnak içi metin) dayanır, bu yüzden cihazda offline çalışabilir. \b regex kelime sınırı kullanmadık: JavaScript'te \b yalnızca ASCII harflerini "kelime karakteri" sayar, bu yüzden çünkü/için/öğretmen gibi ç/ü/ş/ö içeren kelimelerde sessizce eşleşmeyi kaçırır — bu yüzden düz includes() daha güvenilir. Beş ROCKS bileşeninden her biri için üç dilde (TR/DE/EN) kelime listesi tutulur; eşleşen bileşen sayısı arttıkça su tüketimi 30ml'den 10ml'ye düşer (bileşen başına 4ml). Sayfanın en üstündeki Canlı Demo tam olarak bu kodu çalıştırıyor.

rocksAnalyzer.jsJavaScript
const ROLE_KEYWORDS = {
  en: ["as a", "as an", "act as", "you are a", "you are an", "acting as", "imagine you are"],
  de: ["du bist ein", "du bist eine", "als ein ", "als eine ", "in der rolle", "übernimm die rolle"],
  tr: ["sen bir", "senin rolün", "gibi davran", "rolünde"],
};
const ROLE_PATTERNS = [/\bsen\b(?:\s+\S+){0,3}\s+bir\b/i];

const OBJECTIVE_KEYWORDS = {
  en: ["write", "explain", "summarize", "create", "list", "compare", "translate", "solve", "design"],
  de: ["schreibe", "erkläre", "fasse zusammen", "erstelle", "liste", "vergleiche", "übersetze", "löse"],
  tr: ["yaz", "açıkla", "özetle", "oluştur", "listele", "karşılaştır", "çevir", "çöz"],
};

const CONTEXT_KEYWORDS = {
  en: ["because", "for my", "for a", "since", "given that", "so that", "grade", "years old"],
  de: ["weil", "für mein", "für meine", "damit", "da ich", "klasse", "jahre alt"],
  tr: ["çünkü", "için", "dolayı", "nedeniyle", "sınıf", "yaşında"],
};

const KNOWLEDGE_KEYWORDS = {
  en: ["about", "topic"],
  de: ["thema", "konu"],
  tr: ["hakkında", "konusunda"],
};
const KNOWLEDGE_PATTERNS = [/"[^"]{3,}"/, /\d{1,4}/];

const STYLE_KEYWORDS = {
  en: ["bullet points", "as a list", "short", "briefly", "simple language"],
  de: ["stichpunkten", "kurz", "einfach erklärt", "kindgerecht"],
  tr: ["madde madde", "kısaca", "basit bir dille", "kolay bir dille"],
};

const WATER_ML_MAX = 30;
const WATER_ML_MIN = 10;
const COMPONENT_COUNT = 5;
const ML_PER_COMPONENT = (WATER_ML_MAX - WATER_ML_MIN) / COMPONENT_COUNT;

function containsAny(text, keywordsByLang) {
  const lower = text.toLowerCase();
  return Object.values(keywordsByLang).some((list) => list.some((kw) => lower.includes(kw)));
}

export function analyzePromptROCKS(promptText) {
  const text = (promptText || "").trim();

  const matched = {
    role: containsAny(text, ROLE_KEYWORDS) || ROLE_PATTERNS.some((p) => p.test(text)),
    objective: containsAny(text, OBJECTIVE_KEYWORDS),
    context: containsAny(text, CONTEXT_KEYWORDS),
    knowledge: containsAny(text, KNOWLEDGE_KEYWORDS) || KNOWLEDGE_PATTERNS.some((p) => p.test(text)),
    style: containsAny(text, STYLE_KEYWORDS),
  };

  const score = Object.values(matched).filter(Boolean).length;
  const waterMl = Math.max(WATER_ML_MIN, WATER_ML_MAX - score * ML_PER_COMPONENT);
  const ecoCoins = Math.round((WATER_ML_MAX - waterMl) / 2);
  const missing = Object.keys(matched).filter((key) => !matched[key]);

  return { score, matched, waterMl, ecoCoins, missing };
}

export function applyGardenImpact(garden, analysis) {
  const isHealthy = analysis.score >= 4;
  const clarityDelta = isHealthy ? 3 : -5;
  const growthDelta = isHealthy ? 1 : 0;

  return {
    ...garden,
    riverClarity: Math.min(100, Math.max(0, garden.riverClarity + clarityDelta)),
    treeGrowth: Math.min(100, garden.treeGrowth + growthDelta),
    ecoCoins: garden.ecoCoins + analysis.ecoCoins,
  };
}

const ROBBY_MESSAGES = {
  tr: {
    role: "Robby'nin bir ROLÜ yok! 'Sen bir tarih öğretmenisin' gibi bir cümleyle başla.",
    objective: "AMAÇ belirsiz. Net bir fiil kullan: yaz, açıkla, özetle...",
    context: "BAĞLAM eksik. Bunu neden istediğini ya da kim için olduğunu ekle.",
    knowledge: "BİLGİ eksik. Konuyla ilgili somut bir detay veya sayı ver.",
    style: "STİL belirsiz. Cevap kısa mı olsun, madde madde mi?",
  },
  de: {
    role: "Robby vermisst eine ROLLE! Zum Beispiel: 'Du bist ein Geschichtslehrer'.",
    objective: "Das ZIEL ist unklar. Nutze ein klares Verb: schreibe, erkläre, fasse zusammen...",
    context: "Der KONTEXT fehlt. Sag, wofür oder für wen du das brauchst.",
    knowledge: "Das WISSEN fehlt. Gib ein konkretes Detail oder eine Zahl dazu.",
    style: "Der STIL ist offen. Kurz? In Stichpunkten?",
  },
  en: {
    role: "Robby can't find a ROLE! Try: 'You are a history teacher'.",
    objective: "The OBJECTIVE is fuzzy. Use a clear verb: write, explain, summarize...",
    context: "CONTEXT is missing. Say why you need this or who it's for.",
    knowledge: "KNOWLEDGE is missing. Add one concrete detail or number.",
    style: "STYLE is open. Short? Bullet points?",
  },
};

export function getRobbyFeedback(missing, lang = "tr") {
  if (missing.length === 0) return null;
  const dict = ROBBY_MESSAGES[lang] || ROBBY_MESSAGES.tr;
  return missing.map((key) => dict[key]);
}

React'a bağlama

RobbyOverlay.jsxReact
function useRocksWaterMeter(promptText, lang) {
  return useMemo(() => {
    const analysis = analyzePromptROCKS(promptText);
    const feedback = getRobbyFeedback(analysis.missing, lang);
    return { ...analysis, feedback };
  }, [promptText, lang]);
}

function RobbyOverlay({ promptText, lang = "tr" }) {
  const { waterMl, score, feedback, ecoCoins } = useRocksWaterMeter(promptText, lang);
  const isHealthy = score >= 4;

  if (!promptText) return null;

  return (
    <div className={`robby-card ${isHealthy ? "robby-good" : "robby-warn"}`}>
      <strong>{waterMl} ml</strong> su harcanacak
      {!isHealthy && feedback && (
        <ul>
          {feedback.slice(0, 2).map((line) => (
            <li key={line}>{line}</li>
          ))}
        </ul>
      )}
      {isHealthy && <span>+{ecoCoins} Eco-Coin kazandın! 🌳</span>}
    </div>
  );
}
Genişletme notu — applyGardenImpact(), Modül 1'in çıktısını doğrudan Modül 2'nin (My Digital Garden) durum güncellemesine bağlar: her prompt olayı hem su sayacını hem nehir/ağaç durumunu tek bir çağrıda günceller, ayrı bir senkronizasyon adımına gerek kalmaz.
